Activate the cc-web-design skill to review frontend code.

## Quick Usage

Review specific files:
```
/cc-web-design src/components/Button.tsx
```

Review all components:
```
/cc-web-design src/components/
```

Review and create fix tasks:
```
/cc-web-design src/ --create-tasks
```
