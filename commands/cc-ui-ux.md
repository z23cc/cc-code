Activate the cc-ui-ux skill for UI/UX design decisions.

## Quick Usage

Design a new page:
```
/cc-ui-ux design a SaaS dashboard
```

Review existing UI:
```
/cc-ui-ux review src/components/
```

Pick a style + colors:
```
/cc-ui-ux glassmorphism dark mode for e-commerce
```

Get font pairing:
```
/cc-ui-ux font pairing for a tech blog
```
