Activate the cc-browser skill for browser automation.

## Quick Usage

Navigate and screenshot:
```
/cc-browser open https://example.com and take a screenshot
```

Test a form:
```
/cc-browser fill the login form at https://myapp.com
```

E2E test:
```
/cc-browser test the checkout flow on https://store.com
```

Responsive check:
```
/cc-browser check responsive design at https://mysite.com
```
